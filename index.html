Crystal Segmentation — MaSEL Lab

U-Net deep learning model for real-time crystal detection on microscopy images · UIC Research

ML · CV Research

Built a full U-Net segmentation pipeline in PyTorch for the UIC Department of Materials and Systems Engineering. Trained on 300+ manually annotated crystal microscopy images labeled with LabelMe, achieving 80%+ Dice accuracy on unseen test images — a meaningful result in a domain with scarce labeled data.

Automated the preprocessing pipeline in Python (flip, rotation, blur augmentations via torchvision), then developed an OpenCV + C++ post-processing stage to count and visualize detected crystals — cutting manual researcher analysis time by 50%.
